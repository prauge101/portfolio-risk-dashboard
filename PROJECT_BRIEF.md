# Project Brief

## Planned Dashboard Features

- Load stock price data from CSV files
- Calculate daily returns
- Calculate cumulative returns
- Measure volatility
- Estimate drawdown
- Apply portfolio weights
- Calculate historical Value at Risk
- Run a simple Monte Carlo simulation
- Display charts in a Streamlit dashboard

## Tools Used

- Python for the main programming language
- pandas for tabular data handling
- numpy for numerical calculations
- matplotlib for charts
- Streamlit for the dashboard interface
- pytest for testing calculation functions

## Learning Goals

- Build a clean Python project structure
- Separate user interface code from calculation logic
- Learn core portfolio risk concepts
- Practise writing readable, tested functions
- Create a finance project that is simple enough to explain but realistic enough to be useful
